Chat Widget Demo

This page embeds the Botnoi Chat Widget with a 3D avatar using a single <script> tag. Click the chat icon in the bottom-right corner to start a conversation. 

<script src="https://webavatar.didthat.cc/chat-widget.js"
    data-bot-id="YOUR_BOT_ID"
    data-bnv-key="YOUR_BNV_KEY"
    data-bnv-version="1 or 2"
    data-bnv-speaker="SPEAKER_ID"
    data-avatar-url="AVATAR_URL">
</script>

The widget lazy-loads both the chat connection and avatar on first open, generates voice replies via Botnoi Voice TTS, and keeps everything alive when you collapse the panel.
